# Hikmicro / GUI Button Automation

Automatización de botones GUI mediante reconocimiento por imagen (template matching),
pensado para aplicaciones sin API (ej: Hikmicro).

## Características
- Detección por OpenCV (matchTemplate)
- Click físico controlado
- Logging en CSV
- Preparado para ejecución 24/7

## Requisitos
- Windows
- Python 3.9+
- Pantalla activa (no bloqueada)

## Instalación
```bash
pip install -r requirements.txt
